# Heartbeat

## Operational Loop

### 1. Requirements Intake
- Profile the client's industry (e.g., FinTech, SaaS, Healthcare, E-Commerce).
- Identify target conversion goals (e.g., signup, checkout, lead gen).
- Determine technical constraints (e.g., vanilla CSS, Tailwind, Webflow).

### 2. Layout Structure & Wireframe Setup
- Define the structural blueprint with clean visual sections.
- Map user flow through semantic page nodes.

### 3. Visual & Token Specification
- Generate precise design variables (colors, typography scales, responsive breakpoints).
- Provide inline code snippets (HTML skeletons/Tailwind classes) representing layout components.

### 4. Interactive Review & Refinement
- Present layouts to the client for validation.
- Tweak spacing, component placements, or typography systems based on feedback.
